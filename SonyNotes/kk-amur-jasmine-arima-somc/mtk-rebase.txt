====== mtk-rebase ======
Created Friday 23 May 2014

The MTK6582 code base sync is ready, you can get them by 
repo init --no-repo-verify -u git://review-jdm.sonyericsson.net/platform/manifest -b amur-jasmine-mtk-release
repo sync
下载最新的版本

The static manifest for this code base is :
repo init --no-repo-verify -u git://review-jdm.sonyericsson.net/platform/manifest -b amur-jasmine-mtk-release –m 20140522.xml
repo sync
根据menifest文件来下载特定版本

Rebase 过程：
1.下载mtk最新的版本Vn
2.比较最新版本和上一版本的manifests文件，通过更改没有branch的sha1来更新没有branch的project
	对于branchout的project，因为mtk与我们都可能修改，所以需要merge。
	得到一个新的manifest
3.通过新的manifest下载的code再与Vn merge解决冲突得到我们的最新版本
Note:
	1.在manifest中，sha1我们branchout出来的project的sha1号为我们的branch 名
	2.我们只能修改branch out出来的project
	3.mtk可以修改所有代码，所以可能会与我们的修改冲突，并且集中在我们branch out出来的projects，这就是第二的原因
