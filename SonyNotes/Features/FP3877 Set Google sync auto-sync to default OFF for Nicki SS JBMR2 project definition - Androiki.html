[edit] ASW Feature Definition - FP3877 Set Google sync auto-sync to default OFF for Nicki SS JBMR2 project definition

This document should take max 3-6 days to create. Its main purpose is to get a rough time and man power estimate. It will be reviewed by the Distribution Architect.

[edit] Requirements

The high-level requirements for this feature that will be used for the definition:

  • At first start of the phone the setting for ‘Google synch’ shall be off (Settings/Data Synchronization/Auto-synch)
  • This shall NOT be customizable


Reference One Slider: none

[edit] AUI Light or updated Blueprint from UI design

Follow the UI of Email solution for Lagan1.2

[edit] Configuration Requirement

[edit] SIA from M1

None

[edit] Rough Estimate

  • Set auto-sync to off as default.
    • Technical description: Create a broadcast listener which turns off auto sync after the user has signed in to his/her google account.
    • Effort (50%/90%): 3/6 day(s)


  • Total time for this feature (including integration meeting) is
    • 50% 4 man days
    • 90% 7 man days
