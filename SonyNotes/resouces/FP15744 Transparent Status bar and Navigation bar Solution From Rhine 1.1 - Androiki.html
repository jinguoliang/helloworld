[edit] Requirements (SIA)

Feature aims to have a semi-transparent status bar in Lockscreen and Home, suppressed and hidden navigation bar in (a fullscreen) Lockscreen and semitransparent navigation bar in Home. The HTC One X has a semi transparent status bar (but no soft key navigation). The intention is to enlighten the potential buyer of the big screen.

TransparentHome1.png
TransparentHome2.png
TransparentLockscreen.png


Some notes about the scope

The requirements for this feature speak only about Lockscreen and Home, hence those are the main targets. The proposed solution, however, is a general API that can be used by any SOMC application. So if other applications (Camera or Album, for instance) find the new API useful, they are welcome to use it too.
